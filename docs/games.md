# Games built with SUCC

Games shipping or in development using the SUCC character controller.

- **[SurfsUp](https://store.steampowered.com/app/3454830/SurfsUp/)** by **[Mark Arneman](https://bearlikelion.com/)** & **[Nerdiful](https://nerdiful.itch.io/)** - the surf/bhop game whose controller SUCC was extracted from.

## Add your game

Shipped or working on something built with SUCC? [Open an issue on GitHub](https://github.com/bearlikelion/SUCC/issues/new) with:

- Game name and link (Steam, itch.io, website, etc.)
- Author(s) and a link to your profile
- Optional: a one-line description

Your game will be added to this list.
